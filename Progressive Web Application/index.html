Natural Disaster Preparedness PWA

Web Application Description

The web application will provide the user with information on the natural disasters, the different categories of disasters, and the ways on preparing for and adapting to these disasters. A quiz can also be taken by users which will evaluate their knowledge of the various disasters described within the web application.

Features:

Disaster Preparedness Information

Different categories of disasters are described in detail such as the potential hazards they bring and the harm and devastation a calamity can inflict. The resources needed and the methods for preparing and adjusting to the conditions during these events are also described in the web application.

Preparedness Quiz

A quiz can be taken by users to test their knowledge of the information that is covered by the web application. The answers will be stored client-side using IndexedDB.

Push Notifications

Push notifications will simply welcome the user once subscribed and send messages encouraging to read content provided by the web application. Notifications can also encourage users to take the preparedness quiz if they haven't already.
